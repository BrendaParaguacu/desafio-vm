# ☁️ Desafio de Máquinas Virtuais na Azure

## 🚀 Visão geral

Este projeto tem como objetivo consolidar os conhecimentos adquiridos sobre **Máquinas Virtuais na Microsoft Azure**, aplicando na prática conceitos fundamentais de computação em nuvem.

Infelizmente por ser pago acabei por não conseguir por em prática mas durante o laboratório, foi realizada a criação e configuração de uma VM no ambiente Azure, além da documentação completa do processo no GitHub.

---

## 🎯 Objetivos de aprendizagem

✔ Aplicar conceitos de computação em nuvem na prática  
✔ Criar e gerenciar máquinas virtuais na Azure  
✔ Documentar processos técnicos de forma clara e organizada  
✔ Utilizar o GitHub como ferramenta de versionamento e portfólio  

---

## ☁️ Conceitos estudados

- Máquinas Virtuais (VMs)  
- Infraestrutura como Serviço (IaaS)  
- Escalabilidade e elasticidade  
- Gerenciamento via Portal Azure e CLI  
- Boas práticas de segurança na nuvem  

---

## 🛠️ Etapas realizadas

- Acesso ao Portal da Azure  
- Criação de uma Máquina Virtual (Windows)  
- Configuração de recursos (CPU, memória e disco)  
- Definição de usuário e senha  
- Inicialização da VM  
- Testes de acesso e validação  
- Documentação do processo no GitHub
